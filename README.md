# 🎨 Ai-to-pptx
Ai-to-pptx是使用AI技术来自动生成PPTX，并且支持在线修改和导出PPTX。
主要功能:
- 1 使用DeepSeek等大语言模型来生成大纲
- 2 生成的内容允许用户再次修改
- 3 生成PPTX的时候可以选择不同的模板
- 4 支持在线修改PPTX的文字内容，样式，图片等
- 5 支持导出PPTX，PDF，PNG等多种格式
- 6 支持用户设置单独的LOGO和相关背景图片,打造用户专属的PPTX风格
- 7 支持用户设计自己的模板上传到共享平台,分享给其它人使用
